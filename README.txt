Запуск плагина
Для того, чтобы плагин появился появился в программе QGIS, необходимо
 А. Установить плагин FlexGIS
 Б. Уcтановить и настроить плагин Plugin Reloader

2 варианта установки плагина FlexGIS :
Вариант 1. Установка через ZIP
    1. Заархивировать папку flexgis со всем содержимым в zip-архив
    --ВНИМАНИЕ-- Если в конечном архиве будут mac скрытые файлы - ничего не получится
    2. Зайти в QGIS 
    3. Нажать на верхней панеле программы "Установка и управление плагинами"
    4. В появившемся окне в левой панели выбрать "Установка через ZIP"
    5. Указать путь к zip архиву
Вариант 2. Руками положить папку в нужную директорию
    1. Открыть программу QGIS
    2. Нажать на верхней панеле программы Processing 
    3. Выбрать Toolbox
    4. В открывшемся виджете нажать на 2ю слева иконку Python
    5. В появившемся окне выбрать Open existing script
    6. Откроется директория со скриптами, заканичвающася на processing/scripts/тут_скрипты
    7. Перейти на уровень папки processing и найти папку python
    8. В директорию, оканчивающуюся на python/plugins/ положить папку flexgis со всем содержимым
    9. Перезапустить QGIS

Установка плагина Plugin Reloader
    1. Зайти в QGIS 
    2. Нажать на верхней панеле программы "Установка и управление плагинами"
    3. В левом боковом меню выбрать Все плагины
    4. Вбить в поиске Plugin Reloader
    5. Нажать Install plugin
    6. Закрыть меню Управления плагинами
    7. На панели инструментов появится плагин Plugin Reloader
    8. Нажать на стрелку рядом с ним и нажать Настроить
    9. Выбрать плагин FlexGIS

Чтобы отлаживать код необходимо:
    1. Работать в папке плагина, которая лежит в директории, откуда QGIS берет плагины
    2. Вносить изменения в код -- сохранять код -- нажимать кнопку Plugin Reloader
    3. Для удобства все print-ы выводятся в консоль Python в QGIS
